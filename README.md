# cje-extensions

Extensions to [Causal Judge Evaluation](https://arxiv.org/abs/2512.11150).

- [`cvar_v5/validation_study.tex`](cvar_v5/validation_study.tex) — main validation note for Direct CVaR-CJE. It covers the known-DGP validation, transport audit behavior, bootstrap overlay, and the one-seed Arena diagnostic on real policy responses.
- [`cvar_v5/`](cvar_v5/) — current CVaR-CJE implementation and validation workspace.
- [`cvar_v2/`](cvar_v2/) — earlier Direct CVaR-CJE prototype on real Arena data, with cross-fit transport audit, semi-synthetic Monte Carlo validation, n-scaling sweep, and a LaTeX appendix.
- [`cvar/`](cvar/) — initial demo on synthetic data (notebooks).
- [`arxiv_source/`](arxiv_source/) — LaTeX source of the CJE paper.
