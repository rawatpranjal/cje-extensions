# cje-extensions

Extensions to [Causal Judge Evaluation](https://arxiv.org/abs/2512.11150).

- [`cvar_v2/`](cvar_v2/) — Direct CVaR-CJE on real Arena data, with cross-fit transport audit, semi-synthetic Monte Carlo validation, n-scaling sweep, and the LaTeX appendix.
- [`cvar/`](cvar/) — initial demo on synthetic data (notebooks).
- [`arxiv_source/`](arxiv_source/) — LaTeX source of the CJE paper.
